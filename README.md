# Lagerverwaltung

## Projektbeschreibung
Ein webbasiertes Lagerverwaltungssystem mit einem .NET-Backend und einem React-Frontend.  
Das System bietet Funktionen zur Verwaltung und Anzeige von Artikeln.

---

## Funktionen
1. **Artikeldaten anzeigen**: Liste der Artikel mit Menge und Lagerort.
2. **API-Dokumentation**: Swagger-UI zur Visualisierung und Interaktion mit der API.
3. **Trennung von Backend und Frontend**: Flexible Entwicklung und Deployment.
